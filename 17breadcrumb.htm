Display Breadcrumbs

Excerpt

We all know from fairy tales that breadcrumbs show the way home.

The breadcrumb navigation is used to give the user a list of links to navigate in the page hierarchy. Often it is easier to use this kind of navigation than using a given menu where all pages are reachable.

The links provided here are reflecting the hierarchy of the web site and may be created together with the sitemap and menu to give a consistent virtual map of the side or application.

This navigation should not replace the back button in the browser that users will use to go back to where they come from. The browseralso provides a list of the last visited urls in the tab when you press the back button for a longer time.

How it works

The CSS code for this implementation is implemented very similar to the tag-list but now in a horizontal layout using arrows as a metaphor.

  • All links defined inside a BREADCRUMBS container are formatted as inline-blocks.
  • All but the first is extended using the :before pseudo element.
  • All are extended using the :after pseudo element.
  • All sizes are font-size relative
  • CSS variables are used to define the colors in a central place.
  • The :hover effect is changing the css variable to apply the hover color to the background and extensions.
